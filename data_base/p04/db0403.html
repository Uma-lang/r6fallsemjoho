テーブル定義の操作

(1) テーブル名の変更

テーブル名を変更したい場合、ALTER TABLE 旧テーブル名 RENAME AS 新テーブル名 構文を用います。 

mysql> alter table list1b rename as list1x;
mysql> show tables;
+----------------+
| Tables_in_iwak |
+----------------+
| list1          |
| list1a         |
| list1x         |
+----------------+
3 rows in set (0.00 sec)

mysql> select * from list1x;
+------+-------+------+
| cid  | cname | cage |
+------+-------+------+
|    1 | M1    | NULL |
|   11 | m11   | NULL |
|    1 | NULL  | NULL |
|   11 | NULL  | NULL |
+------+-------+------+
4 rows in set (0.00 sec)

(2) テーブルの削除

テーブルを削除したい場合、DROP TABLE テーブル名 を用います。 

mysql> drop table list1x;
mysql> show tables;
+----------------+
| Tables_in_iwak |
+----------------+
| list1          |
| list1a         |
+----------------+
2 rows in set (0.00 sec)
