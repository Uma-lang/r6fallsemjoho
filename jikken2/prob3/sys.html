import sys とは

import sys は、Pythonの標準ライブラリである sys モジュールをインポートするための構文です。このモジュールを使用することで、Pythonインタプリタやその環境に関連する様々な機能を利用できます。

主な用途と機能

  • コマンドライン引数の処理:

    sys.argv を使用して、スクリプト実行時にコマンドライン引数を取得できます。

    import sys
print(sys.argv)

    このコードは、スクリプト実行時に渡された引数をリスト形式で表示します。

  • 標準入力・標準出力のリダイレクト:

    sys.stdin, sys.stdout, sys.stderr を使用して、標準入力、標準出力、標準エラー出力を操作できます。

  • インタプリタの終了:

    sys.exit() を使用して、Pythonプログラムを任意のタイミングで終了させることができます。

    import sys
sys.exit(0)  # 正常終了
  • Pythonの実行環境に関する情報:

    sys.version や sys.platform を使用して、Pythonのバージョンや実行プラットフォームの情報を取得できます。

例

以下のように sys モジュールを使用して、コマンドライン引数を利用したプログラムを書くことができます。

import sys

def main():
    # コマンドライン引数のチェック
    if len(sys.argv) != 2:
        print("Usage: python script.py ")
        sys.exit(1)

    argument = sys.argv[1]
    print(f"Argument passed: {argument}")

if __name__ == "__main__":
    main()

このように、sys モジュールはPythonの実行環境やプログラム実行に関する便利な機能を提供します。
