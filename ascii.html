ASCIIコードを利用したプログラミング

1. ASCIIコードとは

ASCIIコードは、文字を数値で表現する規格です。例えば、'A' は 65、'B' は 66、'a' は 97 などのように、各文字に一意の数値が割り当てられています。C++ では、文字を整数に暗黙的に変換することができ、これを利用してさまざまな操作を行うことができます。

2. 文字のインデックスを利用する例

次のコードは、文字列 S の各文字の位置を配列 X に格納する方法を示しています。このコードは、S[i] の文字が 'A' から何文字目にあるかを計算しています。

#include <iostream>
#include <vector>
#include <string>
using namespace std;

int main() {
    string S = "ABCDEFGHIJKLMNOPQRSTUVWXYZ";
    vector<int> X(26);

    // 文字のインデックスを計算して格納
    for (int i = 0; i < 26; ++i) {
        X[S[i] - 'A'] = i;
    }

    // 結果を表示
    for (int i = 0; i < 26; ++i) {
        cout << "文字: " << S[i] << " -> インデックス: " << X[i] << endl;
    }

    return 0;
}

3. コードの説明

このコードでは、S の各文字から 'A' の ASCII コードを引くことで、各文字が 'A' から何文字目にあるかを計算しています。

重要なポイント

  • ASCIIコードの引き算: S[i] - 'A' の結果は、文字 S[i] が 'A' から何文字目にあるかを示します。
  • 例: 'C' - 'A' は 2 です。したがって、アルファベットの 'C' は 'A' から 2 文字目であることがわかります。
注意: このコードは、S に 'A' から 'Z' の文字が含まれていることを前提としています。異なる文字が含まれている場合、期待しない結果になる可能性があります。

4. 応用例: 文字の出現回数をカウント

次のコードは、文字列内のアルファベットの出現回数をカウントする例です。

#include <iostream>
#include <vector>
#include <string>
using namespace std;

int main() {
    string S = "HELLO WORLD";
    vector<int> count(26, 0);

    // 各文字の出現回数をカウント
    for (char c : S) {
        if (c >= 'A' && c <= 'Z') {
            count[c - 'A']++;
        }
    }

    // 結果を表示
    for (int i = 0; i < 26; ++i) {
        if (count[i] > 0) {
            cout << "文字: " << static_cast<char>('A' + i) << " -> 出現回数: " << count[i] << endl;
        }
    }

    return 0;
}

5. 応用例の説明

このコードでは、文字列 S に含まれるアルファベットの各文字の出現回数をカウントしています。ASCII コードを利用することで、文字のインデックスを簡単に計算し、それを配列のインデックスとして利用しています。

ポイント

  • 大文字のアルファベット 'A' から 'Z' のみを対象にしています。
  • ASCII コードの引き算によって、対応するインデックスを計算し、出現回数を配列に格納しています。
